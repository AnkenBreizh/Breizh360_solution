# Index des interfaces exposées — Breizh360

> **Dernière mise à jour : 08 / 01 / 2026**

But : permettre à une équipe de trouver rapidement **où est documenté le contrat** à consommer, sans lire le code.

## Où trouver les contrats

- **Domaine** → `Breizh360.Domaine/interfaces.md`
- **Données** → `Breizh360.Data/interfaces.md`
- **Métier** → `Breizh360.Metier/interfaces.md`
- **API Métier** → `Breizh360.Api.Metier/interfaces.md`
- **Passerelle** → `Breizh360.Gateway/interfaces.md`
- **Interface utilisateur** → `Breizh360.UI/interfaces.md`
- **Tests** → `Breizh360.Tests/interfaces.md`

## Règle de consommation

- Si une interface n’est pas documentée dans `interfaces.md` du fournisseur : **elle n’existe pas** (ou n’est pas encore stable).
- Si un contrat est ambigu : ouvrir une demande dans `docs/requests.md`.
